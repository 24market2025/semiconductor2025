Electrically Erasable Read Only Memory Market Overview
Electronically erasable read-only memory (EEPROM) is a semiconductor storage device that can be electronically rewritten multiple times. It has the characteristic that the information on the chip can be erased and programmed multiple times without the need for ultraviolet irradiation or removal. The four working modes of EEPROM include read mode, write mode, erase mode and verification mode.
